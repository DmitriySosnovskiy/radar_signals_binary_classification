# radar_signals_binary_classification

Классификацию между камнями (R) и металлическими цилиндрами (M) на основе данных об отражении сигналов радара от поверхностей.

60 входных значений показывают силу отражаемого сигнала под определенным углом. Входные данные нормализованы и находятся в промежутке от 0 до 1.

Графики и данные после обучения и валидации:

loss: 0.1853

accuracy: 0.9358

val_loss: 0.3621

val_accuracy: 0.8095
